In this upcoming post I’ll show you how to make blobby shapes out of polygons. Click or tap on the shape below to see what I mean.

To transform the polygon into a blob we’ll make of quadratic Bezier curves. Don’t worry though, you don’t really need to know the math behind Bezier curves in order to use them. If you’re interested in that, you can refer to the bezier curve wikipedia article.

Oh look here’s a wild Bezier curve now! If you don’t like the way it looks, that’s fine. You can click or tap it to get a new one.

What’s cool is you only need 3 parts to define a quadratic Bezier curve.

  1. a start point
  2. an end point
  3. a control point

Here’s another Bezier curve where you can drag around each of these points.

If you put the start point and end point on the same line and then drag the control point across the line’s midpoint really fast, it kinda looks like you’re strumming a one-string guitar.

The neat thing about Beziers is that you can string them together to make really curvy shapes. Here’s a heart made up of Bezier curves. It’s so big, that it almost doesn’t fit in the viewport. Click or tap it to make it go ba-dum ba-dum.

So the question is, what’s a good strategy for transforming a pointy polygon into a beautiful blob composed of bezier curves? Well, I thought you’d never ask.

Here’s a super secret look at what’s really going.

This is one of those things that’s easier to describe as an algorithm so that’s what I’m gonna do.

for each pair of consecutive midpoints, m1 and m2, of the lines that define the polygon, draw a quadratic bezier curve from m1 to m2 with the curve’s control point being the vertex of the polygon that lies between m1 and m2

This works cuz each Bezier starts where one leaves off giving a smooth blobby shape in the end. It might be helpful to look at a single piece of the curve.
